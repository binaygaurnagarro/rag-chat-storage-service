# RAG Chat Storage Microservice

A production-ready backend microservice for storing chat histories generated by a
Retrieval-Augmented Generation (RAG) chatbot system.

---

## Tech Stack

- Java 21
- Spring Boot 3
- PostgreSQL
- Spring Data JPA
- Bucket4j (Rate Limiting)
- Docker & Docker Compose
- Swagger / OpenAPI

---

## Features

- Create and manage chat sessions
- Store chat messages with optional retrieved context
- Rename sessions
- Mark sessions as favorite
- Delete sessions with cascade delete
- Pagination support for messages
- API Key based authentication
- Rate limiting
- Centralized error handling and logging
- Health check endpoint

---

## API Authentication

All APIs require the following header:
--header 'X-API-KEY: secret-key'
